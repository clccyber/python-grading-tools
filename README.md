# Python Grading Tools

Automated grading tools for Python programming assignments (CIT 142).

## Installation

Clone this repository into your course repo:
```bash
git clone https://github.com/YOUR-USERNAME/python-grading-tools.git tools/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

## Update

To get the latest version:
```bash
cd tools
git pull
cd ..
```

## Usage

Grade an assignment:
```bash
cd chapter2/ex05/student
python ../../../tools/grade.py
```
